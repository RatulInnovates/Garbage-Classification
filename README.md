# Garbage Classification Project

This project focuses on multi-class garbage classification using Deep Learning and Transfer Learning models.

The system classifies waste into different categories such as:
- Battery
- Biological
- Cardboard
- Clothes
- Glass
- Metal
- Paper
- Plastic
- Shoes
- Trash
